npm update常用命令使用
一、更新
　　npm-check检查更新
　　　　　　npm install -g npm-check
　　　　　　npm-check
　　2.　　npm-upgrade更新
　　　　　　npm install -g npm-upgrade
　　　　　　npm-upgrade
　　3.　　更新全局包：
　　　　　　npm update <name> -g
　　4.　　更新生产环境依赖包：
　　　　　　npm update <name> --save
　　5.　　更新开发环境依赖包：
　　　　　　npm update <name> --save-dev
